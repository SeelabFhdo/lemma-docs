    Static Analysis Library

    LEMMA's documentation is currently work in progress. Therefore, to learn more about the design of the static analysis library and the 20 metrics it provides for assessing the quality of LEMMA-based microservice models, please refer to Section 9.5 of the dissertation that conceived LEMMA1.
